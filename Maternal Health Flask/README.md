\# 🤰 Maternal Health Risk Prediction using Artificial Intelligence



\## 📌 Project Overview



Maternal Health Risk Prediction is an Artificial Intelligence based healthcare web application developed using Flask and Artificial Neural Networks (ANN). The application predicts the maternal health risk level using six important clinical parameters collected from pregnant women.



The system classifies patients into:



\- 🟢 Low Risk

\- 🟡 Mid Risk

\- 🔴 High Risk



The project is developed for educational and research purposes.
